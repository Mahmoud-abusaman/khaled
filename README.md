# Coffee Corner - نظام الكاشير الذكي

نظام نقاط البيع (POS) لإدارة المقهى مع تخزين البيانات في ملفات JSON.

## المميزات

- ✅ نقطة بيع سهلة الاستخدام
- ✅ إدارة المنيو والأصناف
- ✅ تقارير المبيعات اليومية
- ✅ تخزين البيانات في ملفات JSON
- ✅ واجهة عربية بالكامل

## التثبيت

```bash
npm install
```

## التشغيل

### الطريقة الأولى: تشغيل الواجهة والخادم معاً (موصى بها)

```bash
npm run dev:full
```

هذا الأمر سيشغل:
- الواجهة الأمامية على http://localhost:3000
- خادم API على http://localhost:3001

### الطريقة الثانية: تشغيل كل جزء بشكل منفصل

**نافذة طرفية 1 - الخادم:**
```bash
npm run server
```

**نافذة طرفية 2 - الواجهة:**
```bash
npm run dev
```

## البيانات المخزنة

يتم تخزين جميع البيانات في مجلد `data/`:

- `data/menu.json` - قائمة الأصناف والمنتجات
- `data/invoices.json` - الفواتير المسجلة

يتم إنشاء هذه الملفات تلقائياً عند تشغيل الخادم، وتحديثها فوراً عند إجراء أي تغييرات.

## النسخ الاحتياطي

لعمل نسخة احتياطية من البيانات، قم بنسخ مجلد `data/` بالكامل.

## استعادة البيانات

لاستعادة البيانات، قم بنسخ ملفات JSON إلى مجلد `data/`.

## المتطلبات

- Node.js 18 أو أحدث
- npm

## البناء للإنتاج

### 1. بناء الواجهة الأمامية

```bash
npm run build:full
```

هذا الأمر سيقوم ببناء الواجهة الأمامية وإنشاء مجلد `dist/` يحتوي على الملفات الجاهزة للنشر.

### 2. تشغيل الخادم في وضع الإنتاج

```bash
npm start
```

سيقوم الخادم بـ:
- تشغيل API على http://localhost:3001
- تقديم الواجهة الأمامية من مجلد `dist/`
- الوصول للتطبيق كاملاً عبر http://localhost:3001

**ملاحظة:** في وضع الإنتاج، يعمل كل من الواجهة الأمامية والخادم على نفس المنفذ (3001).
